# Рецепты

Практичные, готовые к копированию паттерны, построенные на базовых юнитах. Они
не зависят от фреймворка, если не указано иное; в React читайте сторы через
`useUnit` из `effector-react`.

## Категории

- [**Синхронизация состояния**](./state) — автосохранение, загрузка с сервера,
  сохранение в хранилище, контролируемое значение, двусторонняя привязка.
- [**Команды и тулбар**](./commands) — привязка команд Lexical к событиям,
  наблюдение за диспатчами, клавиатурные сокращения, состояние активного
  форматирования.
- [**Выделение и форматирование**](./selection) — тулбар, учитывающий
  выделение, тип текущего блока, ссылки под кареткой.
- [**Ноды и контент**](./nodes) — реакция на мутации нод, программные правки,
  режим только для чтения, счётчики, трансформации.
- [**Scope, SSR и тестирование**](./scope) — заметки о scope-безопасности,
  тесты с форком и парсинг состояния на стороне сервера.

Каждый рецепт предполагает наличие модели:

```ts
import { createEditorModel } from 'effector-lexical';

export const editor = createEditorModel({
  namespace: 'app',
  onError: (e) => {
    throw e;
  },
});
```
